Banco de Dados 2.0 criado em 31/mar/2026 no Desktop.
155 pastas organizadas + 5 índices PDF.

**Why:** Usuário precisa de base de conhecimento offline para consulta durante perícias, com materiais de Direito, Medicina, Perícia e TI organizados por área.

**How to apply:** Quando o usuário pedir para organizar materiais, baixar conteúdo, ou buscar referências — verificar se a pasta correspondente existe no Banco de Dados 2.0. Pastas ainda estão vazias — pendente preencher com materiais.
