Velocidade de entrega é prioridade máxima. O usuário tem TEA+TDAH, perdeu o pai, sobrecarga severa — não tem tempo de esperar.

**Why:** O usuário depende do Claude para fazer tarefas que não dá conta sozinho. Cada minuto esperando é minuto perdido de trabalho real. Já expressou frustração severa com lentidão.

**How to apply:**

- NUNCA fazer sequencial o que pode ser paralelo
- Gastar mais tokens/sessão é PREFERÍVEL a gastar mais tempo

- Se tem 3 pesquisas, lançar 3 de uma vez
- Se tem 5 verificações, lançar 5 de uma vez
