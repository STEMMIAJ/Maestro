Quando o usuário diz "salva essa busca", salvar o resultado da pesquisa de produto em arquivo .md dentro de ~/Desktop/Compras Apartamento/ na categoria correspondente.

**Why:** Usuário quer registro organizado das pesquisas para decisão de compra do apartamento novo.

**How to apply:**
- Pasta raiz: ~/Desktop/Compras Apartamento/
- Categorias: Pet, Cozinha, Banheiro, Quarto, Sala, Limpeza, Eletronicos, Diversos
- Formato: tabela com produto, preço, loja, nota, avaliações + recomendação + orçamento x2 se aplicável
- Criar novas categorias se necessário
- Nome do arquivo: descritivo em kebab-case
