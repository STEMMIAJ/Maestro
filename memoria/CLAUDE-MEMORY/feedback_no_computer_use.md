NUNCA propor computer-use (mcp__computer-use__*) como solução ou fallback. É lento, invasivo e desnecessário — o usuário já automatizava tudo há meses sem essa ferramenta.

**Why:** 16/abr/2026 usuário explicitou com irritação: "EU JA AUTOMATIZAVA ESSA PORRA HA MESES, ESQUECE COMPUTER USE ME UFILHO, ISSO É FIRULA DE AGORA". Sobrecarga cognitiva alta, não tolera lentidão. Histórico anterior também indicava evitar sem autorização explícita.

**How to apply:** resolver por CLI: Bash, scripts, SMB, SSH, prlctl, AppleScript via `osascript`, shared folders do Parallels, Shared Profile (pastas do Windows espelhadas no Mac), arquivos .bat em pasta monitorada. Esgotar todas as vias programáticas antes de reportar bloqueio técnico. Nunca oferecer computer-use como alternativa.
