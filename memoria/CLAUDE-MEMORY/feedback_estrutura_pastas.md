Em qualquer trabalho que envolva mais de 1 arquivo gerado, SEMPRE criar estrutura de pastas organizada E arquivo mestre detalhado.

**Why:** Usuário é autista + TDAH, esquece entre sessões. Arquivos soltos = perde. Estrutura padrão reduz carga cognitiva. Arquivo mestre detalhado = substitui memória.

**How to apply:**
1. Raiz do projeto SEMPRE em pasta nomeada (nunca solta na Mesa)
2. Subpastas padrão por tipo: `scripts/`, `dados/`, `logs/`, `docs/`, `dashboards/`, `config/`
3. Arquivo mestre no topo: `README.md` ou `MAPA.md` com:
   - O que o projeto faz (1 parágrafo)
   - Estrutura de pastas (árvore comentada)
   - Como rodar cada script
   - Arquivos principais e o que contêm
   - Próximos passos
4. NÃO misturar tipos de arquivo na mesma pasta
5. NÃO deixar README curto — ele substitui a memória do usuário

**Exceção:** script único isolado não precisa de estrutura — só se vira projeto.
