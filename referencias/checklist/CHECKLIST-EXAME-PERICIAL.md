## BLOCO 5 — EXAME ESPECÍFICO POR PATOLOGIA

### Ortopédica / Traumatológica
- [ ] Inspeção da região afetada (deformidade, edema, atrofia, cicatriz)
- [ ] Palpação (pontos dolorosos, crepitação, derrame)
- [ ] Goniometria ativa e passiva (registrar graus)
- [ ] Força muscular MRC por grupo
- [ ] Perimetria comparativa bilateral
- [ ] Manobras especiais pertinentes (especificar)
- [ ] Exame do membro contralateral para comparação

### Neurológica
- [ ] Pares cranianos (ao menos II, III, V, VII)
- [ ] Força muscular MRC — 4 membros
- [ ] Sensibilidade tátil, dolorosa, vibratória
- [ ] Reflexos profundos (bicipital, tricipital, patelar, aquileu)
- [ ] Reflexos patológicos (Babinski, Hoffman, clônus)
- [ ] Coordenação (dedo-nariz, calcanhar-joelho)
- [ ] Equilíbrio (Romberg)
- [ ] Marcha especial (tandem, pontas, calcanhares)
- [ ] Sinais meníngeos (se indicado)

### Psiquiátrica
- [ ] Aparência e comportamento
- [ ] Consciência e orientação
- [ ] Atenção e concentração
- [ ] Memória (imediata, recente, remota)
- [ ] Humor e afeto
- [ ] Pensamento (forma e conteúdo)
- [ ] Percepção (alucinações)
- [ ] Linguagem
- [ ] Juízo crítico
- [ ] Pragmatismo (rotina diária, autonomia)
- [ ] Risco suicida avaliado

### Cardiovascular
- [ ] Ausculta cardíaca (ritmo, bulhas, sopros)
- [ ] Pulsos periféricos
- [ ] Edema de MMII
- [ ] Turgência jugular
- [ ] Hepatomegalia
- [ ] Classe funcional NYHA definida

### Reumatológica
- [ ] Articulações acometidas listadas
- [ ] Sinais inflamatórios (calor, rubor, edema)
- [ ] Deformidades articulares
- [ ] Mobilidade articular
- [ ] Força de preensão (se mãos)
- [ ] Nódulos reumatoides / tofos
