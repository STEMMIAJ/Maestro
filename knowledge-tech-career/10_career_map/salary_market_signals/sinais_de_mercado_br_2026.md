# Sinais de Mercado Tech Brasil — 2026

Orientação para negociar, planejar e validar expectativas. Valores são sinais, não promessas. Oscilam com senioridade real, cidade, modalidade (CLT vs PJ vs exterior), stack.

## Fontes recomendadas

- **Glassdoor BR**: dados autorreportados, base grande mas ruído alto.
- **Kenoby Salary Report**: pesquisa anual empresarial.
- **Revelo Report**: tendências tech BR.
- **Nucleomind**: salários tech remote.
- **Gupy**: dados por vaga.
- **Levels.fyi** (exterior, aplicado a BR remoto): big tech e scale-ups.
- **Hired.com**: foco internacional, aplicável a BR remoto.
- **Trampos.co**: criativo/tech.
- **Pesquisa de Salários Stack Overflow (BR cut)**: declínio da base BR, cuidado.

## Modalidades

1. **CLT BR**: encargos altos, benefícios, estabilidade, salário menor em cash.
2. **PJ BR**: cash maior, sem benefícios, risco e custo próprio.
3. **PJ remoto para exterior (USD/EUR)**: cash significativamente maior (2-5x), sem FGTS/INSS, risco cambial e tributário.
4. **Contractor via Deel/Remote/Oyster**: ponte administrativa para modalidade 3.

Ao comparar salários, normalizar sempre para a mesma modalidade.

## Modificadores

- **Remoto para EUA/UE**: multiplicar por 2-4x em USD/EUR.
- **Big tech BR (Nubank, iFood, Mercado Livre, Stone, PagBank)**: topo + equity.
- **Startup early-stage**: floor baixo + equity incerta.
- **Consultoria**: rate mais alto, sem benefício.
- **Cidade**: SP/RJ/Florianópolis puxam para cima. Interior reduz 20-40%.
- **Inglês**: fluente = porta para remoto exterior, multiplicador real.

## Indicadores qualitativos de valor

- **Portfólio público**: GitHub ativo, artigos, palestra → +10 a +25% na oferta.
- **Certificações reconhecidas**: AWS/GCP + nicho → passaporte a entrevistas, não salário direto.
- **Contribuição open source relevante**: assinatura de maturidade.
- **Especialização de nicho**: saúde + IA, segurança financeira, payments — paga prêmio.

## Como negociar

1. Não dizer número primeiro.
2. Pesquisar range para o papel + senioridade + cidade.
3. Pedir range da empresa.
4. Ancorar no topo justificado por portfólio.
5. Negociar total comp: salário + PLR + equity + benefícios + flexibilidade.
6. Comparar ofertas simultâneas quando possível.
