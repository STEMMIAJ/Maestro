# Prontidão para Certificações

Avaliação honesta de % de prontidão atual para certificações relevantes ao perfil (médico-perito + automação pericial + dados de saúde). Decisão: **perseguir** (fazer em janela definida), **adiar** (fazer quando pré-requisitos existirem), **descartar** (não serve ao perfil).

## 1. CS50x (Harvard — Introduction to Computer Science)
- **Objetivo**: base formal de CS. Alicerce para lacunas de C, algoritmos, memória.
- **Pré-requisitos**: inglês lido; 10h/semana por 10–12 semanas.
- **Prontidão atual**: ~40% (Python OK; C, SQL avançado, JS — ausentes).
- **Decisão**: **perseguir** — iniciar em 2026-Q3, concluir em 2027-Q1. Valor alto para base formal.

## 2. CS50 Python (CS50P — curso standalone)
- **Objetivo**: formalizar Python básico-intermediário.
- **Pré-requisitos**: inglês lido.
- **Prontidão atual**: ~70% (falta OOP e testing).
- **Decisão**: **perseguir** — concluir em 2026-Q2 (8 semanas). Rápido, confirma base.

## 3. Linux Foundation LFS101 (Introduction to Linux)
- **Objetivo**: base de Linux além do macOS/zsh diário.
- **Pré-requisitos**: terminal confortável.
- **Prontidão atual**: ~60%.
- **Decisão**: **perseguir** em 2026-Q3 — gratuito, 2 semanas.

## 4. Google Data Analytics Professional Certificate (Coursera)
- **Objetivo**: selo reconhecido em análise de dados + SQL + Tableau.
- **Pré-requisitos**: nenhum formal. 8 cursos, ~6 meses part-time.
- **Prontidão atual**: ~45% (SQL e visualização são gaps).
- **Decisão**: **adiar** — primeiro cobrir SQL e pandas (P1 do backlog). Reavaliar em 2026-Q4.

## 5. AWS Cloud Practitioner (CLF-C02)
- **Objetivo**: introdução à AWS, selo básico.
- **Pré-requisitos**: nenhum. Prova paga ~USD 100.
- **Prontidão atual**: ~15% (zero prática cloud).
- **Decisão**: **adiar** — só faz sentido se decidir migrar para cloud. Reavaliar em 2027-Q1.

## 6. CompTIA Security+
- **Objetivo**: selo de segurança foundational reconhecido pelo mercado.
- **Pré-requisitos**: 1–2 anos de experiência em TI (flexível).
- **Prontidão atual**: ~20%.
- **Decisão**: **adiar** — avaliar após cobrir lacuna ALTA #5 (auditoria LGPD + segurança aplicada). Reavaliar em 2027-Q2.

## 7. DataCamp Data Analyst (career track)
- **Objetivo**: trilha prática de data analyst.
- **Pré-requisitos**: assinatura DataCamp.
- **Prontidão atual**: ~50%.
- **Decisão**: **descartar** — sobreposto com CS50P + Google Data Analytics. Escolher um só (Google).

## 8. MIT 6.0001 (Introduction to Computer Science and Programming in Python)
- **Objetivo**: OCW gratuito. Base formal.
- **Pré-requisitos**: nenhum.
- **Prontidão atual**: ~65%.
- **Decisão**: **adiar** ou **descartar** — sobreposto com CS50P. Fazer só se CS50P não satisfazer.

## 9. Anthropic Prompt Engineering / Claude Developer Skills (curso oficial)
- **Objetivo**: selo em prompt engineering + SDK Claude.
- **Pré-requisitos**: nenhum.
- **Prontidão atual**: ~75% (uso intensivo).
- **Decisão**: **perseguir** — rápido, alinhado ao trabalho atual. Concluir em 2026-Q2.

## 10. HL7 FHIR Foundation — Proficiency (self-paced)
- **Objetivo**: certificação em padrão de interoperabilidade de saúde.
- **Pré-requisitos**: conhecer FHIR R4.
- **Prontidão atual**: ~10%.
- **Decisão**: **adiar** — depende do item 15 do backlog. Reavaliar em 2027-Q2.

## 11. ICPI / IAPP CIPM ou CIPP/E (privacidade)
- **Objetivo**: selo formal em gestão de privacidade / LGPD.
- **Pré-requisitos**: conhecimento regulatório (o perito já tem base jurídica).
- **Prontidão atual**: ~40%.
- **Decisão**: **perseguir** em 2027 — casa com perfil pericial + automação com dado sensível. Alto valor diferencial.

## 12. Certificação CNJ — Perito Judicial Digital (quando/se existir)
- **Objetivo**: formalizar competência pericial em meio digital.
- **Pré-requisitos**: nomeação judicial ativa.
- **Prontidão atual**: ~85% (é exatamente o que o perito faz).
- **Decisão**: **perseguir** assim que CNJ publicar edital. Monitorar.
